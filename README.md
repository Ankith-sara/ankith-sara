## Projects

**[Demand Forecasting System](https://github.com/Ankith-sara)**  
Hybrid SARIMA + LSTM model for retail demand prediction in Indian markets. Combines statistical interpretability with deep learning accuracy to optimize inventory planning and reduce wastage. ~82% forecasting accuracy on real retail datasets.

**[AI Short Video Generator](https://github.com/Ankith-sara)**  
Text prompt → rendered 1080p video via a serverless async pipeline. Built with Next.js, Remotion, Convex, and Inngest. Handles script generation, scene composition, and background rendering without manual editing.

**[AI Website Builder](https://github.com/Ankith-sara)**  
Natural language → fully responsive website layout. Next.js App Router + Convex backend + AI-driven component generation. Structured output, not templates.

**[AI Tour Planner](https://github.com/Ankith-sara)**  
Prompt-to-itinerary travel app built with React Native, Gemini AI, Firebase, and Google Maps. Generates day-by-day trip plans with flights, activities, and destination images.

**[Full Stack E-Commerce Platform](https://github.com/Ankith-sara)**  
React + Node.js + MongoDB. Auth, cart, payment integration, and admin dashboard.

**[AI-Based Authentication System](https://github.com/Ankith-sara)**
